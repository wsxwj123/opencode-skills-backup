# 符号表模板（Symbol/Notation Table）

> 适用场景：理工科论文含大量数学/物理变量时必设；纯生命科学临床类可选。
> 符号表与缩略语表独立，不得合并：缩略语表列英文缩写，符号表列数学/物理符号。

## 使用说明

1. 在 `atomic_md/符号表.md` 中按以下模板填写，merge 时与缩略语表一起纳入前置页。
2. 符号按**类别分组**（如拉丁字母变量、希腊字母变量、下标、运算符），组内按字母升序。
3. 无该章节内容时删除对应行，勿保留空行占位。

## Markdown 模板

```markdown
## 符号表

### 拉丁字母变量

表 0-1：拉丁字母变量符号说明

| 符号 | 含义 | 单位 |
|---|---|---|
| $A$ | 截面积 | m² |
| $C$ | 浓度 | mg/mL |
| $D$ | 扩散系数 | m²/s |
| $E$ | 弹性模量 | GPa |
| $F$ | 力 | N |
| $k$ | 速率常数 | s⁻¹ |
| $L$ | 长度 | m |
| $m$ | 质量 | g |
| $n$ | 摩尔数 | mol |
| $P$ | 压力 | Pa |
| $Q$ | 流量 | mL/min |
| $R$ | 气体常数 | J/(mol·K) |
| $T$ | 温度 | K 或 °C |
| $t$ | 时间 | s |
| $V$ | 体积 | mL 或 L |

### 希腊字母变量

表 0-2：希腊字母变量符号说明

| 符号 | 含义 | 单位 |
|---|---|---|
| $\alpha$ | 热扩散系数 | m²/s |
| $\beta$ | 修正系数 | — |
| $\varepsilon$ | 孔隙率 | — |
| $\eta$ | 动力黏度 | Pa·s |
| $\lambda$ | 热导率 | W/(m·K) |
| $\mu$ | 化学势 | J/mol |
| $\rho$ | 密度 | kg/m³ |
| $\sigma$ | 应力 | MPa |
| $\tau$ | 剪切应力 | MPa |
| $\phi$ | 体积分数 | — |

### 下标说明

表 0-3：常用下标含义

| 下标 | 含义 |
|---|---|
| 0 | 初始值或参考值 |
| eff | 有效值 |
| exp | 实验值 |
| max | 最大值 |
| sim | 模拟值 |
```

## 注意事项

- **不得**将已在缩略语表中列出的英文缩略语（如 ELISA、PCR）重复放入符号表。
- 符号使用 LaTeX 行内公式语法（`$符号$`），`markdown_to_docx.py` 渲染为公式字体。
- 若项目使用 CSU 默认格式，符号表放在缩略语表之前（前置页顺序：缩略语表 → 符号表，或根据学校要求调整）。
- 单位统一采用 SI 制；无量纲用 `—`。
