# Assets 目录说明

本目录用于存放辅助资源文件。

## 图表占位符说明文件

当论文生成过程中使用图表占位符时，会在 `04_图表文件/` 目录下生成对应的 JSON 说明文件。

### 示例文件结构

```json
{
  "figure_id": "1-1",
  "title": "全球癌症发病率与死亡率统计（2010-2020）",
  "description": "柱状图展示2010-2020年全球癌症新增病例和死亡病例的变化趋势",
  "source": "数据来源：WHO 2021年报告",
  "format": "建议使用双Y轴柱状图，X轴为年份，左Y轴为发病率，右Y轴为死亡率",
  "data_reference": "可从 WHO 官网下载原始数据",
  "placeholder": "[图 1-1：全球癌症发病率与死亡率统计（2010-2020）]"
}
```

## 使用流程

1. 论文生成时，在正文中插入占位符：`[图 X-Y：图片标题]`
2. 系统自动生成对应的 JSON 说明文件
3. 用户根据说明文件制作实际图表
4. 在 Word 中将占位符替换为真实图片

## 注意事项

- 所有图表必须符合中南大学格式要求
- 图片建议使用 300 DPI 以上分辨率
- 表格建议使用三线表样式
- 确保所有图表都有规范的题注
