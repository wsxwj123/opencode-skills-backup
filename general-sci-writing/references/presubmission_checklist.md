# 投稿前作者自检清单（soft 提醒，不阻断交付）

> 本清单由技能在终稿门禁前以 soft 方式提示。机器无法可靠裁决的项（需作者掌握原始数据/图像/外部工具），一律只提醒、不阻断。
> 已被本技能其他 hard 门禁覆盖的维度（引用一一对应、去AI、跨节一致、伦理批号、统计严谨等）不在此重复。

## A. 数字与一致性
- [ ] **摘要中的数字（p 值、n、效应量、百分比）与正文逐一对应**，无矛盾（高频 desk-reject）。
- [ ] 标题/摘要/关键词与正文主题一致；关键词不与标题词完全重复。

## B. 语言规范
- [ ] **英式/美式拼写全文统一**（colour↔color、analyse↔analyze 不混用）——技能扫描器会给出混用提示，作者择一统一。
- [ ] 缩写全文首次定义、之后统一使用；摘要内独立首展；Title 不用缩写（DNA/RNA/PCR 等通用词除外）。

## C. 图表与原始数据（机器无法核，作者必查）
- [ ] **图像无不当处理**：自查 6 类——Western blot 条带像素重复、图像区域克隆、不当切割拼接（边界线/亮度跳变）、亮度对比度选择性调整、跨实验复用图像（旋转/翻转/裁剪）、柱状图与原始数据表不一致。
- [ ] 误差线类型（SD/SEM/95%CI）在图注中定义且与正文一致；统计图报告 n 值；显微镜图有 scale bar。
- [ ] **Source Data**（Nature/Cell 强制）：每张主图一个 sheet，含全部原始数据点，数值与图对应。
- [ ] 图像分辨率达期刊要求（印刷图通常 ≥300 DPI）。

## D. 学术诚信（需外部工具/作者声明）
- [ ] **文本相似度查重**：投稿前用 iThenticate 或期刊工具检查，相似度通常 <15–20%。
- [ ] **自我抄袭（text recycling）**：复用自己已发表的方法段落须改写或注明；说明与已有预印本的关系。
- [ ] **AIGC 使用声明**：按目标刊要求声明是否使用 AI 写作辅助；禁止将 AI 列为作者。

## E. 合规与注册（缺则桌面拒）
- [ ] **临床试验注册号**（NCT/ChiCTR/ISRCTN 等）：前瞻性临床研究须在首例入组前注册（ICMJE 强制）；回顾性注册须坦诚声明。
- [ ] 数据可用性声明（DAS）：存储库名称/DOI/accession number 完整；不公开须说明理由。
- [ ] 代码/软件可用性：自编代码开源链接（GitHub/Zenodo/OSF）；商业软件注明版本号。

## F. 报告规范附件（按研究类型）
- [ ] RCT→CONSORT 流程图+清单；观察性→STROBE；动物→ARRIVE 2.0（essential 10）；系统综述/Meta→PRISMA 2020+PROSPERO 注册号。流程图与 checklist 作为投稿附件一并提交。

## G. 投稿材料齐全
- [ ] Cover letter（significance + 3 个 key findings + 适配目标刊理由）。
- [ ] 推荐/回避审稿人（推荐附邮箱，近 3 年无共同发表/共同基金/同单位/师生关系）。
- [ ] Highlights（Cell 系，3–5 条，每条 ≤85 字符）/ One-sentence summary（Nature 系）/ Graphical abstract（按刊要求）。
- [ ] 正文字数、主图数量在期刊上限内；补充材料（SI）图表编号连续、格式与主文一致。
