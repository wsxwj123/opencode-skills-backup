# 试卷排版规范

## 通用排版规则

### 试卷头部

```
{学校名称}
{学年}学年{学期}
{年级}{学科}（{考试类型}）试卷
（满分：{总分}分  时间：{时长}分钟）

学校__________ 班级__________ 姓名__________ 考号__________
```

示例：
```
长沙市第一中学
2025-2026学年第一学期
九年级语文（期末考试）试卷
（满分：120分  时间：120分钟）

学校__________ 班级__________ 姓名__________ 考号__________
```

### 题号编排

- **大题**：中文数字 + 顿号 → 一、二、三、四
- **大题标题格式**：`一、积累运用（20分）` —— 大题名称后括号标注该大题总分（长沙中考三大题：一、积累运用20分；二、阅读50分；三、写作50分）
- **小题**：阿拉伯数字 + 点 → 1. 2. 3.（全卷连续编号，不按大题重新计数）
- **小小题**：括号编号 → （1）（2）（3）

### 分值标注

- 每小题分值标注在题干末尾：`（2分）`
- 有多个小小题时，在小小题后标注：`（1）XXXX（2分）`
- 大题总分在大题标题后标注

### 答题空间

- **选择题**：题号后留括号 `（    ）` 或方框
- **填空题**：下划线 `__________`，长度适中
- **简答题**：留 3-5 行横线或空白
- **翻译题**：留 2-3 行横线
- **作文**：标注字数要求，留足够方格纸空间（Word 中用提示文字替代）

### 字体与间距（对标 2025 长沙中考真题原卷版，make_paper.py 已内化，无需手设）

- **大标题**：宋体，三号（16pt），**加粗**，居中
- **副标题**（满分/时间）：宋体，小四（12pt），居中
- **注意事项 / 大题标题 / 小标题**：宋体，小四（12pt），**加粗**，左对齐
- **正文 / 题干 / 选项**：宋体，五号（10.5pt）
- **阅读选文正文**：**楷体**，五号，首行缩进 2 字符
- **选文标题 / 作者**：楷体，居中
- **出处标注**：宋体，五号，**右对齐**
- **行距**：1.5 倍行距
- **页边距**：上 1.61cm，下 2.54cm，左右 1.91cm（A4 纵向）

> 注：真题原卷标题与各级标题用的是**宋体加粗**（非黑体），选文正文用**楷体**。
> 此前文档误记为黑体，已据真题校正。

### 分隔与排列

- 大题之间空一行
- 阅读材料用 `material` 块呈现，选文正文楷体缩进，与题目自然区分
- **古诗词整体居中**（标题/作者/诗句逐行居中）：`material.layout="verse"`
- 文言文/小说/散文正文楷体首行缩进 2 字；篇名居中；末尾出处右对齐
- 选择题选项：A/B/C/D 各选项单独一行，左对齐，缩进2字符

## 语文试卷特殊排版

### 阅读材料排版
```
阅读下面的文章，完成 X-X 题。

                    {文章标题}
                                        {作者}

    {正文第一段缩进两格}XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

    {正文第二段}XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

### 古诗文排版
```
阅读下面的文言文，完成 X-X 题。

    {篇名}
    {朝代}·{作者}

    {正文}XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

### 古诗词排版
```
阅读下面的诗歌，完成 X-X 题。

    {诗题}
    {朝代}·{诗人}

    {诗句第一联}，{诗句第二联}。
    {诗句第三联}，{诗句第四联}。
```

### 作文题排版
```
21. 请你以"____"为题，写一篇文章。（50分）

    {材料内容/题目要求}

    要求：①{要求一}；②{要求二}；③{要求三}；④不少于600字。
```

## 参考答案排版

```
{学校名称}
{学年}学年{学期}
{年级}{学科}（{考试类型}）参考答案及解析

一、积累运用（20分）
1.（2分）{答案}　【解析】{错因/依据}
2.（4分）{答案}
...

评分说明：
- 选择题每题只有一个正确答案
- 简答题给采分点（"答出X点给X分，意思对即可"），不拘泥于参考表述
- 作文评分参照中考四等评分标准：一等(45-50) 二等(38-44) 三等(30-37) 四等(0-29)

【教师备注】（可附于答案末尾）
- 考点分布表、难度分析、命题意图、与教材的关联
```

## Word 输出注意事项

- 使用 A4 纸张，纵向
- 页码居中，格式："第 X 页 共 X 页"
- 试卷页和答案页分开（不同文件或分节）
- 避免题目跨页断裂（尤其是阅读材料和对应题目）
- 选择题选项排列整齐，使用制表位对齐
