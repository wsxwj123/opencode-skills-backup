# 去AI感与人情味出题指南

## 一、AI出题的常见病症

### 1. 套话病
```
× "请认真阅读下面的材料，完成后面的题目。"
✓ "阅读下面的文章，完成1-4题。"（简洁直接，和真题一致）

× "同学们，在我们的日常生活中，阅读是一种重要的能力..."
✓ 直接给材料，不加任何引导语废话

× "综合以上材料，请你谈谈对XX的理解和感悟。"
✓ "结合文章第③段，分析作者为什么说'XXXXX'。"（具体、有锚点）
```

### 2. 选项同质化
```
× A. 坚持不懈的精神  B. 勇敢无畏的品质  C. 善良温暖的品格  D. 乐观豁达的态度
  （四个选项结构完全一致，一看就是AI生成）

✓ A. 他性格倔强，不肯认输  B. "老头子"是个有担当的人  C. 作者对故乡的复杂情感  D. 文章批判了冷漠的社会风气
  （长短不一，表述方式各异，像真实教师出的）
```

### 3. 材料鸡汤化
```
× 选取充满正能量、处处有哲理的"心灵鸡汤"文
✓ 选取有生活质感的真实文本：一封家书、一段采访、一篇新闻特写、一个真实故事
```

### 4. 作文题假大空
```
× "以'坚持'为话题，写一篇不少于600字的文章"
× "生活中处处有美，请以'发现美'为题..."
✓ "外婆总说：'这把伞还能用。'请以'还能用'为题，写一篇记叙文。"
✓ "在某个平常的下午，你突然理解了一个人。以此为开头，续写一篇文章。"
```

## 二、人情味注入策略

### 综合性学习题
从学生真实生活场景出发：

```
✓ 学校开展"探访长沙老街"综合性学习活动，你所在的小组负责调查太平街的历史文化。请完成以下任务：
  （1）为本次活动拟一条宣传标语。
  （2）采访太平街一位老店主，请你设计两个采访问题。
  （3）活动结束后，班级要办一期主题板报，请设计两个板块名称。

× "学校开展'弘扬传统文化'主题活动，请你参与。"（太泛，没有具体场景）
```

### 情境式默写
给出具体使用场景，而非死记硬背：

```
✓ 小明在作文中想引用古诗表达对远方朋友的思念，他可以引用王勃《送杜少府之任蜀州》中的名句：_______________，_______________。

✓ 班级辩论赛上，正方需要引用《岳阳楼记》中表达"先忧后乐"思想的句子来支撑观点：_______________，_______________。

× 请默写范仲淹《岳阳楼记》中的名句。（太直白，没有情境）
```

### 阅读题设问
问题要具体、有锚点，避免大而泛：

```
✓ "文中第⑤段画线句'他的手抖了一下'，结合上下文，分析'抖'字表现了父亲怎样的心理。"
✓ "作者在结尾说'我终于读懂了那碗面'，联系全文，说说他'读懂'了什么。"

× "请分析文中人物的心理变化。"（太宽泛）
× "本文表达了作者怎样的思想感情？"（中学生最讨厌的"标准问法"）
```

## 三、真实教师出题特征（模仿目标）

1. **题干简洁**：不废话，直接告诉学生做什么
2. **选项有陷阱**：干扰项反映学生真实的常见错误，不是随便凑的
3. **阅读材料有温度**：选择有故事性、有细节的文本，不选议论文堆砌
4. **默写有情境**：不是"请写出XX的下一句"，而是给出一个使用场景
5. **作文贴近生活**：从一个具体的小切口进入，不是大主题大概念
6. **分值有层次**：简单记忆类2分，理解分析类3-4分，综合鉴赏类4-5分
7. **语言风格统一**：全卷语言风格一致，不会前半部分正式后半部分随意
8. **偶有地方色彩**：综合性学习或阅读材料中融入本地文化元素

## 四、赏析词语题答题模型（供命题参考）

命题时要清楚标准答题步骤，才能出好采分点：

```
第一步：解释词义（本义 + 语境义）
第二步：分析效果（表现的内容，可从词性/修辞/描写/写作手法角度切入）
第三步：表达的情感
```

分析角度：词性（动词/形容词/副词）、修辞手法、人物描写（外貌/语言/神态/动作/心理）、写作手法（对比/衬托）

## 五、作文命题策略（源自教研培训）

### 作文"三步法"
三个情节词 + 三个情感词 + 三次典型描写

### 作文"五笔法"（记叙文）
轻点一笔（交代事件）→ 简叙一笔（略写事件）→ 详写一笔（描画人物）→ 曲写一笔（写出波澜）→ 深化一笔（点明主旨）

### 导语设计原则
- 导语篇幅以三到五句为宜
- 内容不宜过具体，避免罗列句式
- 要启发而非限制学生创作
- 提示语揭示选材方向和写作重点

### 半命题作文设计原则
- 补题部分最好为有抽象性的名词或名词短语
- 避免过于具体的表述，以防限制写作思路
- 给予补题选择空间，同时保持限制性

## 六、自检清单

出题完成后，对每道题逐一检查：

- [ ] 这道题像不像一个教了10年书的语文老师出的？
- [ ] 题干有没有多余的废话？
- [ ] 选项长短是否参差不齐（而非整齐划一）？
- [ ] 阅读材料读起来是否有"被挑选"的感觉，而非"被生成"的？
- [ ] 设问角度是否具体、有锚点？
- [ ] 作文题是否能让学生有话可写、有情可抒？
- [ ] 默写题是否有使用情境？
- [ ] 综合性学习是否有具体的活动场景？
