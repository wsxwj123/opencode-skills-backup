# 数学·物理·化学·生物出题专属要求

> 出理科卷在 Phase 2 动笔前必读本文档。通用命题规则在 SKILL.md「①通用命题规则」。

## 1. 命题规则（理科特有）

- **知识点锚定**：严格按 `meta.textbook`（Phase 1 已确认的教材版本/册次）和课标确定考查范围。
- **解答题须给完整步骤与公式**；评分标准按步骤给分（列式/代入/结果分步采分）。
- **多选题/实验探究题**在每题 `meta.type` 中声明类型。
- **数据真实正确**：题干内嵌数据/公式/实验结果必须自洽且符合物理常识（如 g 取 9.8 或 10 须全卷统一）；**实验数据不得编造不可能的结果**。溯源硬门禁不覆盖纯数据题，须自觉把关。
- **理科真题素材基本无法自动抓**（题库全付费），需用户提供或按知识点生成新题（详见 `material-sources.md`）。

## 2. 选材规格

理科无阅读选文字数门禁；规格要点：

| 项 | 规格 |
|------|------|
| 计算量 | 中考单题计算 ≤3 步链条；压轴题可 4-5 步 |
| 实验题 | 基于教材真实实验改设情境，数据自洽 |
| 情境题 | 真实生活/科技情境（可 fetch_web 抓科普报道做引子，引子落盘 materials/）|

## 3. 配图执行（理科最大约束）

配图限制的锚点定义见 SKILL.md「适用范围⚠️」：本技能**不自动生成图形**，三选一处理——①题干改详细文字描述；②`figure` 块 `src` 用用户提供的图片；③留占位 `［图：…］` 人工补图后交付。

执行规则：
- 严格按 `meta.decisions.figure_plan`（Phase 1 已确认）执行，不得中途自行改方案。
- **build 有缺图硬门禁**：题干含"如图/如下图/图中/图所示"等字样但 paper 块无 figure 块 → 拒绝出卷（exit 2）。确需占位强制出卷加 `--allow-missing-figure`，且必须告知用户"缺图位置为文字占位，学生可能无法作答"。
- `figure.src` 留空时 build 输出 `［图：…］` 占位，交付时必须列入待办清单。

## 4. 理科反模式

| 反模式 | 为什么不行 | 替代做法 |
|--------|-----------|----------|
| 未告知配图限制就开始出题 | 物理/化学/生物大量需配图，学生收到占位符无法作答 | Phase 1 确认科目时必告知配图限制并定 figure_plan |
| 题干写"如图"却不给图 | 废题；build 缺图门禁会拒绝 | 改文字描述或要求用户提供图 |
| 编造实验数据/不可能的结果 | 知识性错误，教师一眼识破 | 数据自洽、符合常识，必要时给出推导 |
| 全卷物理常数不统一 | g=9.8 与 g=10 混用导致答案冲突 | 卷首注明取值，全卷统一 |
| 照搬付费题库原题 | 违反原创底线且无法溯源 | 按知识点+新情境原创命题 |

## 5. Phase 1 需额外确认的参数（理科）

| 参数 | 默认 | 写入 |
|------|------|------|
| 🔴 配图方案（三选一）| 必问，无默认 | `meta.decisions.figure_plan` |
| 教材版本/册次 | 不限 | `--textbook` / meta.textbook |
| 是否含多选/实验探究题 | 按当地卷型 | meta.decisions |
| 物理常数取值（物理）| g=10（初中惯例）| 卷首注意事项 |
