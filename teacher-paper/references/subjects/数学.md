# 数学出题专属要求

> 出数学卷在 Phase 2 动笔前必读本文档。通用命题规则（去AI感/难度控制/能力层次）在 SKILL.md「①通用命题规则」，本文档只写数学特有的部分。

## 1. 命题规则（数学特有）

- **课标锚点**：严格按 `meta.textbook`（如人教版/北师大版/华东师大版）与课标确定考查范围；初中按 2022 版课标"四基四能"（基础知识·基本技能·基本思想·基本活动经验），高中按 2017 版课标六大核心素养（数学抽象·逻辑推理·数学建模·直观想象·数学运算·数据分析）。
- **符号与书写规范**：函数 $f(x)$、集合 $\{x|\ldots\}$、向量 $\vec{a}$/粗体、点 $A(x,y)$、角 $\angle ABC$、相似 $\sim$ 全等 $\cong$ 等符号全卷一致；π 取值（保留 π 或取 3.14）、根号化简到最简、分母不带根号等结果规范须在卷首注明。
- **分步采分铁律**：解答题必须给出"列式 → 代入 → 计算 → 结论"的完整步骤；评分按步骤给分（每步关键采分点 1-2 分），不能只看最终答案。几何证明须"已知→求证→证明"结构齐全，每一步注明依据（定理/公理/已知）。
- **数据真实性**：应用题数据须可信（如总人口、利率、距离），统计题数据来自真实情境（《国民经济统计公报》/教育部公开数据），不得编造离谱数字（如初中生身高 2.5 m）。统计/概率题须自洽（频率和=1、概率∈[0,1]）。
- **真题素材**：中考/高考真题题库基本无法自动抓，本技能按知识点+新情境**原创命题**；可参照教材课后习题与近 3 年真题的命题风格与梯度，但**禁止换皮改编**（换数字/换字母即视为抄袭）。

## 2. 选材规格与图表来源

数学无阅读选文字数门禁；情境引子取自真实数据/科技报道时，须落盘 materials/ 并打 `material.block="情境引子"`（机器豁免逐字校验，但内容须真实）：

| 项 | 规格 |
|------|------|
| 选择题计算量 | 单题 ≤3 步运算 / 高中可 ≤4 步 |
| 解答题链条 | 中考 ≤4 步；高考压轴 ≤5-6 步 |
| 应用题情境字数 | 80-200 字（不含算式）|
| 情境引子来源 | 国家统计局 stats.gov.cn / 教育部 / 中国科普博览 / 中科院科普 |

### 配图来源与做法

| 图类 | 推荐做法 |
|------|----------|
| 函数图象（一次/二次/反比例/三角/导数）| `figure.kind="function"` 自动渲染 |
| 数轴/不等式解集 | `figure.kind="number_line"` |
| 几何图形（三角形/四边形/圆/相似/全等）| `figure.kind="geometry"` 或 AI 直接写 SVG 入 `figure.kind="svg"` |
| 统计图（条形/折线/饼图/散点）| `figure.kind="bar/line/pie/scatter"` |
| 向量/平面解析几何 | `figure.kind="vector"` 或 SVG |
| 立体几何（三视图/直观图）| AI 写 SVG，复杂建议用户提供 |
| 真实情境照片 | 用户提供，存 materials/ |

## 3. 题型逐项命题工艺

### 选择题（初中3分/高中5分）

- **形式**：四选一（高中含多选）。每题考一个核心知识点，干扰项反映学生典型错误。
- **真题特征**：题干 30-60 字，含一个明确数学对象（方程/函数/图形）和一个问题。
- **干扰项陷阱类型表**（必覆盖 2-3 类）：

  | 陷阱类型 | 示例 |
  |----------|------|
  | 漏解/增解 | 二次方程开方时漏负根；绝对值方程未分类讨论 |
  | 定义域/取值范围 | 反比例函数 $x\neq 0$、对数 $x>0$ |
  | 符号错误 | 移项不变号；不等式两边乘负数不变号 |
  | 概念混淆 | 中位数与平均数；众数唯一性；频率与频数 |
  | 单位/换算 | 厘米与米、分与小时 |
  | 特殊值代入正确但一般情形错 | 取 $x=0$ 时成立但全体不成立 |

### 填空题（初中3分/高中5分）

- **形式**：直接填结果，无步骤分。
- **结果规范**：分数化为最简、根号化为最简、π 保留与否全卷统一、多解题"或"分隔（如 $x=2$ 或 $x=-3$）、范围题用区间或不等式（与卷首声明一致）。
- **命题要点**：答案唯一或可枚举；多解须明确说"所有满足条件的值"；新定义/开放题填一个示例即可（须在答案解析列举所有可能）。

### 解答题·计算化简、解方程（中考12分档）

- **采分细则**（每步 1-2 分）：① 列出方程/算式（1 分）；② 正确变形/代入（1-2 分）；③ 运算无误（1-2 分）；④ 写出结论、检验（1 分）。
- **常考类型**：实数运算（含绝对值/根号/三角函数特殊值）、整式分式化简、解一元二次方程、不等式组、二元一次方程组。

### 解答题·几何证明（中考12-17分档）

- **三段式书写**：已知（题干已给）→ 求证（明确写出要证的命题）→ 证明（每步注明依据）。
- **采分细则**：① 正确标注/作辅助线（1-2 分）；② 关键中间结论（如全等/相似/平行）（2-3 分）；③ 推理逻辑无跳步（1-2 分）；④ 最终结论（1 分）。
- **常考定理**：全等三角形 SSS/SAS/ASA/AAS/HL；相似三角形 AA/SAS/SSS；圆的切线判定与性质；平行四边形/特殊四边形判定。

### 解答题·应用题（中考12分档）

- **三要素**：① 真实情境（人口/利率/路程/工程/方案选择）；② 数据可信（标注来源）；③ 设元清晰（"设 X 为…"）。
- **采分细则**：① 设元（1 分）；② 列方程/不等式（2-3 分）；③ 求解（2-3 分）；④ 检验并答（1-2 分，须答出实际意义，如"该商品定价 25 元"）。

### 解答题·压轴题（中考10分/高考13分）

- **梯度分层**：必设 (1)(2)(3) 三小问，难度依次为 0.7 / 0.5 / 0.3；(1) 让中等生拿到 30%-50% 分。
- **中考压轴常考**：二次函数综合（与几何结合）、几何动点（与函数结合）、新定义探究。
- **高考压轴常考**：导数综合（含参讨论/恒成立/零点）、解析几何（直线与圆锥曲线位置关系）、数列综合。
- **命题陷阱**：(2) 分类讨论须穷尽情形；(3) 探究题须给出参考解法，不留无答案的开放题。

## 4. 反模式黑名单

| 反模式 | 为什么不行 | 替代做法 |
|--------|-----------|----------|
| 未告知配图限制就开始出题 | 几何/函数题大量需图，占位符无法作答 | Phase 1 确认配图方案并写入 meta.decisions.figure_plan |
| 题干写"如图"却不给图 | 废题；build 缺图门禁拒绝 | 改文字描述、自动渲染、用户提供三选一 |
| 编造数据/不可能的结果 | 应用题用初中生身高 2.5 m、年利率 50% 等离谱数据 | 数据须真实可查，注明来源 |
| 全卷符号/取值不统一 | π 取 3.14 与保留 π 混用、g/重力单位混用 | 卷首注意事项明确声明 |
| 照搬付费题库原题 / 换皮改编 | 违反原创底线，学生刷过即识破 | 按知识点+新情境原创命题 |
| 解答题只给最终答案 | 学生无法分步得分；评分无依据 | 必给"列式+代入+计算+结论"完整步骤 |
| 压轴题三小问梯度倒挂 | (1) 比 (3) 还难，区分度失效 | 严格按 0.7/0.5/0.3 梯度设计 |
