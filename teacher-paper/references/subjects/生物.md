# 生物出题专属要求

> 出生物卷在 Phase 2 动笔前必读本文档。通用命题规则（去AI感/难度控制/能力层次）在 SKILL.md「①通用命题规则」，本文档只写生物特有的部分。

## 1. 命题规则（生物特有）

- **课标锚点**：严格按 `meta.textbook`（人教版/北师大版/苏教版）和课标确定考查范围。初中按"生物体的结构层次·生物圈中的绿色植物·生物圈中的人·遗传与进化·生物与环境·生物技术"八大主题；高中按必修一（分子与细胞）二（遗传与进化）三（稳态与调节）四（生物与环境）+ 选择性必修。
- **生物术语规范**：
  - 用学名（如"ATP""mRNA""DNA"全大写）；
  - 细胞器名称完整（"线粒体"不写"线粒"，"内质网"不写"内质"）；
  - 生理过程用规范动词（"合成""分解""转运""调节"），避免口语化（"做出""产生"）；
  - 遗传符号：基因用单字母（A/a），基因型连写（AaBb），表现型用文字描述。
- **数据真实性**：实验数据须符合生物学规律（如人体血糖正常 3.9-6.1 mmol/L、体温 36.5-37.5°C、染色体数目按物种实际）；不得编造不可能的实验结果（如"所有酶在 100°C 活性最高"）。
- **图文规范**：曲线图坐标轴须标物理量与单位；遗传系谱图圆形=女、方形=男、实心=患病、空心=正常；细胞结构图各细胞器须可辨。
- **真题素材**：中高考真题题库基本无法自动抓，按知识点+新情境原创命题；可参照近 3 年真题命题风格与梯度，**禁止换皮改编**。

## 2. 选材规格与图表来源

生物无阅读字数门禁；情境引子取自真实科研/科普报道时，须落盘 materials/ 并打 `material.block="情境引子"`（机器豁免逐字校验，但内容须真实可查）：

| 项 | 规格 |
|------|------|
| 选择题信息量 | 题干 30-100 字 |
| 识图题信息量 | 含图（曲线/结构/系谱）+ 50-150 字说明 |
| 实验探究题信息量 | 实验目的+材料+步骤+数据，200-500 字 |
| 综合题信息量 | 真实科研情境引子 100-300 字 + 3-5 问 |
| 情境引子来源 | 中科院科普 / 国家自然博物馆 / 中国植物图像库 / 央视《自然传奇》/ 科普中国 / bioRxiv 公开摘要 |

## 3. 题型逐项命题工艺

### 选择题（2 分/题，初中）（2 分/题，高中）

- **形式**：四选一。生物选择题大量为"图文混合题"（含曲线/结构/系谱图）。
- **干扰项陷阱类型表**（必覆盖 2-3 类）：

  | 陷阱类型 | 示例 |
  |----------|------|
  | 概念混淆 | 细胞分裂 vs 分化、有丝分裂 vs 减数分裂、遗传 vs 变异 |
  | 范围/数量 | "所有""都""只""一定" 等绝对化词 |
  | 因果倒置 | "因为基因突变所以进化"（应为"突变是进化的原料"）|
  | 图象解读错误 | 曲线峰值/拐点/交点意义；坐标轴单位变化 |
  | 实验对照误判 | 对照组 vs 实验组、空白对照 vs 自身对照 |
  | 生理过程时序 | 光合作用光反应/暗反应顺序、有丝分裂前期/中期 |
  | 等位基因/同源染色体混淆 | 基因座位、染色体配对 |

### 填空与简答题（2-4 分/空）

- **形式**：填术语/生理过程名称/结构名称/简要文字解释。
- **采分细则**：术语规范（错字/同音字不给分）；解释类须答出"原因+结果"（如"促进光合作用，因为 CO₂ 浓度升高"）。
- **常考类型**：识图填名称、补全生理过程、解释实验现象。

### 实验探究题（10-16 分）

- **变量控制三要件**：① **自变量**（实验组与对照组差异，须唯一）；② **因变量**（观察/测量的指标，须可量化）；③ **无关变量**（须严格控制相同：温度/pH/光照/时间）。
- **采分细则**：

  | 阶段 | 采分点 |
  |------|--------|
  | 假设 | 合理性（1-2 分） |
  | 分组与变量控制 | 对照设置（2-3 分）；变量识别（2 分） |
  | 实验步骤 | 步骤完整（编号 ①②③，2-3 分） |
  | 预期结果与结论 | 若…则…，由此可知…（2-3 分） |

- **常考实验**（初中）：观察细胞/植物组织、绿叶在光下制造有机物、种子萌发条件、唾液淀粉酶催化作用、光合作用/呼吸作用、植物向光性。
- **高中**：酶的特性探究、光合作用影响因素、植物激素作用、种群密度调查（样方法/标志重捕法）。

### 遗传系谱与概率计算题（10-14 分，高频）

- **采分细则**（三步法）：

  | 步骤 | 采分点 |
  |------|--------|
  | 步骤①判定遗传方式 | 显隐性判断（看子代是否出现新性状）+ 常/性染色体判断（看患病者男女比例）（3-4 分） |
  | 步骤②写出关键个体基因型 | 父母+子女基因型推断（3-4 分） |
  | 步骤③概率计算 | 棋盘法/分支法/乘法定理（3-4 分），过程完整 |

- **结论书写规范**：基因型须写所有可能（如 AA 或 Aa）；概率须化简（如 1/4 不写 25%/0.25 须与卷首统一）。
- **常考类型**：单基因遗传分析、双基因独立/连锁、性别决定与伴性遗传、人类遗传病（系谱图）。

### 综合题（10-16 分，高考）

- **形式**：真实科研情境（如"科学家发现某蛋白调控植物开花"）+ 3-5 问递进。
- **命题要点**：情境来自真实科研报道（中科院/Nature/Cell 公开摘要）；设问由易到难（识别→分析→设计→评价）；至少 1 问考"信息迁移"（用题干新信息推理，不能直接套教材）。
- **采分细则**：每问独立采分；开放性问题须给参考答案要点（"答出 X 点给 X 分"）。

## 4. 反模式黑名单

| 反模式 | 替代做法 |
|--------|----------|
| 未告知配图限制就出题 | Phase 1 确认 figure_plan |
| 题干写"如图所示"却不给图 | 改文字描述/AI 写 SVG/用户提供 |
| 编造实验数据/反生物常识 | 数据须符合生物学规律 |
| 术语口语化/不规范 | 用学名与规范动词 |
| 遗传题不给清亲本基因型 | 亲本基因型必须题干给出或可推 |
| 实验题对照设计缺失 | 必给对照组与无关变量控制 |
| 选择题用"所有""一定"等绝对化词作正确项 | 绝对化词通常为干扰项 |
| 照搬付费题库原题/换皮改编 | 按知识点+新情境原创 |
