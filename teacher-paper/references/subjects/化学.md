# 化学出题专属要求

> 出化学卷在 Phase 2 动笔前必读本文档。通用命题规则（去AI感/难度控制/能力层次）在 SKILL.md「①通用命题规则」，本文档只写化学特有的部分。

## 1. 命题规则（化学特有）

- **课标锚点**：严格按 `meta.textbook`（人教版/沪教版/鲁教版）和课标确定考查范围。初中按"身边的化学物质·物质构成的奥秘·物质的化学变化·化学与社会"四大主题；高中按必修一二与选修内容（反应原理/有机/物质结构），命题前以 `meta.exam_scope` 锁定单元。
- **化学用语规范铁律**：
  - 元素符号大小写正确（Na 不是 NA，CO 不是 Co）；
  - 化学式下角标在元素符号右下（H₂O 不写 H2O）、原子团带括号（Ca(OH)₂）；
  - 离子符号电荷数字在前正负号在后（Mg²⁺ 不写 Mg+2）；
  - **化学方程式三要件**：① 配平（系数最小整数比）；② 等号上方标条件（点燃/加热"△"/催化剂/高温）；③ 气体加 ↑、沉淀加 ↓（仅在反应物全为溶液且生成物有气体/沉淀时才标）。
- **数据真实性**：摩尔质量、相对原子质量、溶解度等数据须查表（人教版教材附录或 PubChem chembl 公开数据库）；不得编造反应（如"Cu + 稀 H₂SO₄"按反应处理）。
- **真题素材**：中高考真题题库基本无法自动抓，按知识点+新情境原创命题；可参照近 3 年真题命题风格与梯度，**禁止换皮改编**。

## 2. 选材规格与图表来源

化学无阅读字数门禁；情境引子取自真实科普/工业/科研报道时，须落盘 materials/ 并打 `material.block="情境引子"`（机器豁免逐字校验，但内容须真实可查）：

| 项 | 规格 |
|------|------|
| 选择题信息量 | 题干 30-100 字 |
| 工业流程题信息量 | 150-400 字（含流程文字描述）|
| 实验探究题信息量 | 实验目的+器材+步骤+现象+数据表，200-400 字 |
| 情境引子来源 | 科普中国 / 中科院化学所新闻 / PubChem（化合物信息）/《中国化学》科普版 |

## 3. 题型逐项命题工艺

### 选择题（3 分/题，含不定项 3 分/题）

- **形式**：四选一（不定项考"全对得满分，选对不全得 2 分，错选得 0 分"）。
- **干扰项陷阱类型表**（必覆盖 2-3 类）：

  | 陷阱类型 | 示例 |
  |----------|------|
  | 偷换条件 | "稀盐酸"误作"浓盐酸"；"常温"误作"高温" |
  | 特殊值反例 | "金属都能与稀酸反应"忽略 Cu/Ag |
  | 单位/状态 | g 与 mol、标准状况下气体摩尔体积 22.4 L/mol |
  | 化学用语错误 | 方程式未配平、漏 ↑↓ 或多加 ↑↓、漏写条件 |
  | 概念混淆 | 化合物 vs 混合物、化学变化 vs 物理变化、燃烧 vs 缓慢氧化 |
  | 反应条件 | "点燃"与"加热"差异、催化剂是否必需 |
  | 守恒陷阱 | 质量守恒只对反应物生成物总质量，不对元素种类外的产物 |

### 填空与简答题（2-6 分/空）

- **形式**：填化学式/名称/化学方程式/简要文字解释。
- **采分细则**：
  - 化学方程式每个 3 分（配平正确 1 分 + 条件正确 1 分 + ↑↓ 正确 1 分；漏一项扣 1 分）；
  - 化学式/名称每空 1-2 分；
  - 简答题须答出"现象 + 原因（反应/原理）"，缺一项扣半。

### 工业流程题（10-15 分，高频考点）

- **形式**：给一个真实工业流程方框图（如海水提溴、铝土矿炼铝、湿法炼铜），设 4-6 问。
- **设问链典型结构**：
  1. 原料预处理目的（粉碎/酸浸/碱浸的作用）；
  2. 关键反应化学方程式书写；
  3. 操作名称（过滤/蒸发/结晶/萃取）；
  4. 杂质处理（沉淀法/调 pH 法）；
  5. 副产品/循环物质识别；
  6. 绿色化学评价。
- **命题要点**：流程须真实可查（参考人教版教材或工业化学手册）；不得自编不存在的工艺。

### 实验探究题（10-12 分）

- **三段式采分**：

  | 阶段 | 采分点 |
  |------|--------|
  | 提出假设/猜想 | 合理性（1-2 分） |
  | 设计与变量控制 | 对照实验设计（2-3 分）；仪器/试剂选择（1-2 分） |
  | 现象/结论书写 | 现象描述（如"生成白色沉淀，气泡逸出"，2-3 分）；结论扣题（2 分） |

- **常考实验**（初中）：气体制取（O₂/CO₂/H₂）、酸碱中和反应探究、金属活动性顺序探究、CO₂ 与 NaOH 反应探究、变质碱液成分探究、燃烧条件探究。
- **高中**：物质制备与性质验证、化学反应原理实验（电化学/平衡）、有机物结构与性质实验。

### 计算题（5-15 分）

- **采分细则**：① 设未知数（1 分）；② 写出化学方程式（2-3 分，含配平条件）；③ 列出比例关系（2 分）；④ 计算（2-3 分）；⑤ 写出答（含单位，1 分）。
- **常考类型**：依方程式定量计算、溶液质量分数/物质的量浓度计算（高中）、混合物组成计算。

### 推断题（10-15 分，初中高频）

- **形式**：给物质转化框图（A→B→C，含"产生气体""加酸生成沉淀"等线索），推断物质。
- **命题要点**：突破口须明确（如"红色固体且能与稀酸反应→Cu/Fe₂O₃"）；线索充分不冗余；答案唯一。

## 4. 反模式黑名单

| 反模式 | 替代做法 |
|--------|----------|
| 未告知配图限制就出题 | Phase 1 确认 figure_plan |
| 题干写"如图装置"却不给图 | 改文字描述/AI 写 SVG/用户提供 |
| 化学方程式不配平/漏条件/漏↑↓ | 评分前自查；建立化学用语审校清单 |
| 编造不存在的反应/工艺 | 反应须可查教材或权威化学文献 |
| 元素符号大小写错误 | 卷面校对至少 2 遍 |
| 推断题答案不唯一 | 突破口须充分，答案路径唯一 |
| 工业流程题脱离实际 | 流程须真实可查（教材或工业手册）|
| 照搬付费题库原题/换皮改编 | 按知识点+新情境原创 |
