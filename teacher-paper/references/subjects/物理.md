# 物理出题专属要求

> 出物理卷在 Phase 2 动笔前必读本文档。通用命题规则（去AI感/难度控制/能力层次）在 SKILL.md「①通用命题规则」，本文档只写物理特有的部分。

## 1. 命题规则（物理特有）

- **课标锚点**：严格按 `meta.textbook`（人教版/教科版/沪科版/粤教版）和课标确定考查范围；初中以"声·光·热·力·电"五大板块为骨架，高中按"力学·电磁·热学·光学·近代物理"五模块（按学考/选考分层）。
- **物理量与单位规范**：SI 制；m/s 不写 m/秒；力的单位 N（不写"牛"在数字旁）；卷面统一用国标符号（速度 v、加速度 a、力 F、电场强度 E、磁感应强度 B）。**全卷符号一致**——重力加速度 g 取 10 m/s² 还是 9.8 m/s²、π 取值，须**卷首注意事项明确声明**，全卷严禁混用。
- **三段式书写**：实验题答题须"现象/数据 → 分析 → 结论"；计算题须"受力分析图 → 列原理式 → 代入数据 → 算出结果（含单位）"。
- **数据真实性**：题中速度/质量/距离/电压等数据须符合物理常识（汽车正常速度 60-120 km/h 而非 500 km/h；干电池 1.5 V 而非 15 V）。实验数据须自洽（误差范围合理，不得得出"摩擦因数=2"等不可能结果）。
- **真题素材**：中高考真题题库基本无法自动抓，按知识点+新情境原创命题；可参照近 3 年真题命题风格与梯度，**禁止换皮改编**。

## 2. 选材规格与图表来源

物理无阅读字数门禁；情境引子取自真实科技/科普报道时，须落盘 materials/ 并打 `material.block="情境引子"`（机器豁免逐字校验，但内容须真实可查）：

| 项 | 规格 |
|------|------|
| 单选题信息量 | 题干 30-80 字，1 个核心情境 |
| 多选题信息量 | 80-120 字，2-3 个判断角度 |
| 计算题情境字数 | 100-250 字（不含算式与已知量）|
| 实验题信息量 | 实验目的 + 器材 + 步骤简述 + 数据表 |
| 情境引子来源 | 科普中国 kepuchina.cn / 中科院科普 / 新华网科技频道 / 央视《新闻直播间》航天报道 |

### 配图来源与做法

| 图类 | 推荐做法 |
|------|----------|
| 受力分析图（矢量箭头）| `figure.kind="vector"` 自动渲染 |
| 运动 v-t / s-t 图 | `figure.kind="function"` 或 `line` |
| 简单光路图（平面镜/折射）| AI 写 SVG 入 `figure.kind="svg"` |
| 简单电路图（串/并联含电表）| AI 写 SVG（仅 3-4 元件级别），复杂电路用户提供 |
| 复杂实验装置（弹簧测力计/天平/伏安法）| 用户提供照片或示意图 |
| 复杂光路（透镜成像/光的色散）| 用户提供 |
| 真实物理现象照片 | 用户提供 |

### 🔴 figure 字段硬声明（v3.22.0 新增）

**问题背景**：以下题型若用纯文字描述视觉关系（"导线下方放磁针"、"线圈在磁场中绕轴转动"、"两板水平放置"），学生无图可参照→题目残废；旧版仅扫题干"如图"字样的门禁会被绕过。

**强制要求**：以下题型在 `items/NN_*.json` 的 `meta` 里**必须**写 `"figure": "required"`，build 时校验缺 figure block→exit 2：

| 题型 | 必声明 figure: required 的场景 |
|------|------------|
| 力学 | 受力分析、斜面/滑轮/连接体、v-t / s-t / a-t 图象题、抛体运动轨迹 |
| 电磁学 | 电路图（串/并联/电桥）、磁场中的导线/线圈方位、带电粒子在电磁场中的轨迹、电磁感应装置 |
| 光学 | 光路图、透镜成像、平面镜反射、折射/全反射 |
| 实验 | 伏安法/打点计时器/天平/弹簧测力计等装置图、纸带/刻度尺读数 |
| 热学/近代物理 | 气缸活塞图、能级跃迁图 |

**figure 块字段结构（v3.24.0 修正——务必照此写，字段直接写在块顶层）**：

| 写法 | figure 块字段 |
|------|------|
| AI 自动渲染 SVG | `{"type":"figure","kind":"svg","svg":"<svg xmlns=\"http://www.w3.org/2000/svg\" ...>...</svg>","alt":"线圈示意","caption":"图1"}` |
| AI 自动渲染函数图 | `{"type":"figure","kind":"function","funcs":["x**2-2*x-3"],"xrange":[-2,4],"alt":"v-t图"}` |
| AI 自动渲染受力矢量 | `{"type":"figure","kind":"vector","alt":"受力分析"}`（详见 make_figure.py 的 vector spec）|
| 用户提供图片 | `{"type":"figure","src":"materials/装置图.png","alt":"实验装置","caption":"图2"}` |

🔴 **三个铁律**：
1. SVG 内容写在 figure 块的 **`svg` 字段**（顶层），**不要**写成 `"spec":"<svg>..."`（spec 当字符串会被降级为占位，图丢失）。
2. 完整 svg 字符串须含 `<svg ...>` 开标签；可省略 xmlns（渲染器自动补）。
3. 自动渲染需本机有 matplotlib（svg 另需 PyMuPDF/fitz）；缺依赖则降级为 ［图：alt］ 占位，build 会在末尾按占位比例 warn。

**完整 meta + figure 示例**：
```json
{"meta": {"num": "9", "score": 6, "figure": "required", "type": "多选"},
 "paper": [
    {"type": "question", "text": "矩形线圈绕水平轴匀速转动..."},
    {"type": "figure", "kind": "svg",
     "svg": "<svg width=\"200\" height=\"120\"><rect x=\"40\" y=\"30\" width=\"120\" height=\"60\" fill=\"none\" stroke=\"black\"/></svg>",
     "alt": "矩形线圈在磁场中", "caption": "图：线圈转动示意"},
    {"type": "options", "items": ["A. ...", "B. ..."]}
 ]}
```

**反模式（必拒）**：把图描述写进 text 字段（"如下图：线圈绕轴..."字符画/ASCII 图）；遇到没法画的图就 `figure: "none"` 后纯文字描述—— 学生看不懂等于残题。

## 3. 题型逐项命题工艺

### 单项选择题（3-4 分/题）

- **形式**：四选一，考一个核心概念或简单计算。
- **干扰项陷阱类型表**（必覆盖 2 类）：

  | 陷阱类型 | 示例 |
  |----------|------|
  | 单位换算 | km/h 与 m/s、kPa 与 Pa、kWh 与 J |
  | 隐含条件 | "光滑"是否给出、"匀速"是否说明、"在真空中" |
  | 极值/特殊情况 | 弹簧最大形变、最高点速度为 0、临界状态 |
  | 参考系混淆 | 火车里看树后退、地面参考系 vs 物体参考系 |
  | 概念混淆 | 质量与重量、压强与压力、电压与电流 |
  | 公式适用条件 | $W=Fs$ 须力与位移同向、欧姆定律仅纯电阻 |

### 多项选择题（4-6 分/题，全对满分/漏选半分/错选 0 分）

- **形式**：四选项含 2-3 个正确。题干信息量大，需综合分析。
- **命题要点**：每个选项独立考一个判断点；正误兼有，避免"全对""全错"；至少 1 个干扰项设"看似正确"陷阱。

### 填空题（2-4 分/题）

- **形式**：填数值（含单位）/物理量名/公式结论。
- **采分细则**：数值正确（1 分）+ 单位正确（1 分）；漏单位扣 1 分；多空题每空独立采分。
- **常见类型**：仪器读数（刻度尺/电表/温度计）、估测题、简单代入计算。

### 实验探究题（6-14 分/题）

- **三段式采分**：

  | 阶段 | 采分点 |
  |------|--------|
  | 变量控制 | 自变量/因变量识别（1-2 分）；控制变量法应用（1-2 分） |
  | 数据处理 | 表格读数/作图描点连线（2-3 分）；计算物理量（2 分） |
  | 结论书写 | "在……情况下，……越大，……越大/不变"句式（2-3 分） |

- **常考实验**（初中）：测密度、测平均速度、探究滑动摩擦力、探究浮力大小、伏安法测电阻、探究电流跟电压电阻的关系、测小灯泡电功率、探究通电螺线管磁极。
- **高考**：力学实验（验证牛顿第二定律/机械能守恒/单摆测重力加速度）、电学实验（伏安法测电阻含分压分流选择/多用电表/传感器）。

### 计算题（10-20 分/题）

- **必带受力分析图**（含图题必填 figure 块，缺图门禁会拒绝）。
- **采分细则**：

  | 步骤 | 分值占比 |
  |------|----------|
  | 受力分析/过程分析（含示意图）| 20-30% |
  | 列出原理公式（牛顿第二定律/动能定理/能量守恒）| 20-30% |
  | 代入数据求解 | 20-30% |
  | 结论（含单位与物理意义）| 10-20% |

- **常考类型**：力学综合（匀变速直线运动+牛顿第二定律+功能关系）、电学综合（电路+焦耳定律+电功）、力电综合（带电粒子在电磁场中运动）。

### 压轴题（10-12 分，高考）

- **梯度分层**：(1) 必给基础分（识别情境、写已知）；(2) 中等分（一个守恒定律应用）；(3) 拉分（多过程/临界/最值）。
- **常考主题**：天体运动+万有引力、电磁感应+能量守恒、带电粒子在复合场中运动。
- **命题陷阱**：不留无答案的开放题；多过程题须明确分段；临界状态题须给提示（如"恰好…"）。

## 4. 反模式黑名单

| 反模式 | 替代做法 |
|--------|----------|
| 未告知配图限制就出题 | Phase 1 确认 figure_plan |
| 题干写"如图"却不给图 | 改文字描述/自动渲染/用户提供三选一 |
| 编造实验数据/不可能结果 | 摩擦因数 0.1-0.8、密度符合物质常识 |
| g 取值全卷不统一 | 卷首声明，严禁混用 |
| 单位制不规范 | SI 制；公式用 m/s 而非 km/h |
| 照搬付费题库原题/换皮改编 | 按知识点+新情境原创 |
| 计算题不给受力分析步骤 | 必给"分析图+原理式+代入+结论"完整步骤 |
| 多选题"全对/全错"型 | 每题 2-3 个正确项，正误兼有 |

## 5. 难度系数锚定表（v3.23.0 新增 — 校准 AI 自评）

> AI 自评难度极易高估：实际 0.45 的题被标"0.7"，导致 Phase 3.5 难度跨度门禁（<0.25 报警）失效。
> Phase 2 设 `difficulty` 字段前对照本表校准。

### 题型 × 难度区间锚定（高考）

| 题型 | 0.85+ 送分 | 0.65-0.85 基础 | 0.45-0.65 中档 | 0.25-0.45 拉分 |
|------|---|---|---|---|
| 单项选择（4分）| 直接概念辨析 | 常规公式代入 | 含 2 个干扰陷阱 | 多概念综合判断 |
| 多项选择（6分）| 不出现 | 单一知识点 2-3 个判断 | 跨知识点综合 | 含强干扰陷阱 |
| 填空/实验题 | 直接读数 | 单步计算/规律识别 | 多步推理/数据处理 | 误差分析/方案设计 |
| 综合计算 | 不出现 | 单一过程/守恒律 | 多过程/分段 | 临界状态/最值 |
| 压轴题 | 不出现 | 不出现 | 不出现 | **必须 ≤0.40** |

### 整卷难度分布参考（高考）

- **基础题** 25-30%（难度 0.65-0.85）
- **中档题** 40-50%（0.45-0.65）
- **拉分题** 20-25%（0.30-0.45）
- **压轴** 5-10%（≤0.35）

⚠️ **自检红线**：
- 压轴题若 >0.40 → 命题陷阱不够，重设参数（让多过程/临界/最值真正起拉分作用）
- 整卷难度均值若 >0.65 → 全卷偏简单，按上述分布重新校准 difficulty 字段
- 难度跨度 <0.25 → Phase 3.5 自审会拦截，必须补足分层
