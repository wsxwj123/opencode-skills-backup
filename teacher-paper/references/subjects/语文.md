# 语文出题专属要求

> 出语文卷在 Phase 2 动笔前必读本文档。通用命题规则（去AI感/难度控制/能力层次）在 SKILL.md「①通用命题规则」，本文档只写语文特有的部分。

## 1. 命题规则（语文特有）

**长沙中考语文结构摘要**：3 大题 21 题，分值 20（积累运用）/50（阅读）/50（写作），满分 120 分、120 分钟；文言文断句选择题固定为第 16 题。**逐题结构、分值、真题实例与各题型详解，以 `references/exam-templates.md` 为唯一权威**，命题前对照其「二、逐题结构表」与「三、各题型详解」。

命题五原则（仿得像·有根·依标·适度·有据）与四特征的完整表述见 `exam-templates.md`「四、命题原则与特征」。

### 语文去AI感补充（通用策略之外）

```
✓ 默写有使用情境（不要赤裸裸“请默写XX”）
✓ 综合性学习有具体活动场景（校运会/班委竞选等真实任务）
✓ 作文题从具体小切口进入，贴近学生真实情感
```

### 作文命题要点

半命题为稳妥之选；补题宜为有抽象性的名词或名词短语；导语 3-5 句启发而非限制；可参考"作文三步法""五笔法"（详见 `anti-ai-guidelines.md`）。字数下限**按学段缩放**（见 SKILL.md 作文规格表），不是一律 600。

### 古诗文命题避坑

翻译选有语境铺垫的句子；简答给要点而非死答案；至少一题考"怎么写"（手法/表达效果）；断句题给四种断法，干扰项错在切错主谓/动宾边界。

### 名著命题避坑

选段须涉关键转折点；选项/问题关联前后情节，不读全书不能答；简答须结合整本书。初中必读书目表见 `exam-templates.md`「六、名著阅读考查范围」；**书目版本须与用户确认**。

## 2. 选材规格与字数区间表

> 本表即 `assemble.py init` 写入 `meta.block_len` 的语文默认区间（九年级规格，其它学段按比例缩放）；build 按此表校验字数。

| 阅读板块 | 字数区间（净字数）| 常见题材/体裁 | 默认来源 |
|----------|----------|---------------|----------|
| 非连续性文本 | 合计 600-1000（2-3 则）| 科技/科普/社会/数据图表 | 人民网、新华网、中科院科普 |
| 现代文·小说 | 1000-1500 | 乡土/成长/亲情/家国/科幻 | 《读者》《意林》或人民网特写 |
| 现代文·散文 | 800-1200 | 写景抒情/记人叙事/哲理 | 同上 |
| 古诗词 | 一首（4-8 句）| 怀古/羁旅/咏物/送别/边塞 | 古诗文网 www.gushiwen.cn、ctext |
| 文言文 | 100-200 节选 | 传记/议论/笔记小说 | 古诗文网《世说新语》《古文观止》 |
| 名著 | 片段 + 全书背景 | 按学段教材必读书目 | 教材必读名著（须确认书目）|

### 选材原则

```
文言文：与教材呼应或对比（同主题/同人物类型/同文体），传记类优先
古诗词：选课外但风格手法呼应课内的作品
小说/散文：推荐《读者》《意林》，有故事性和细节，主题正向但忌说教
非连文本：科普+跨学科+图文混合（《科学之友》《博物》等）
综合性学习：校园生活、社会实践、地方文化的真实任务情境
名著：聚焦核心情节与典型形象，覆盖全书关键转折点，避免烂熟片段
```

> 术语注意：卷面板块名按当地真题原卷（如长沙 2025 原卷作"非文学作品阅读"），但选材性质是**非连续性文本**——多则真实材料组合，出处必标。

## 3. 排版要求

- 选文 material 块的写法与排版铁律（随卷给全文/出处右对齐/标题作者/verse 居中/楷体缩进，**build 有硬门禁**）见 `references/authoring-workflow.md`「选文 material 块」节，不在此重复。
- 古诗词必须 `layout="verse"` 整体居中；小说/散文标题+作者楷体居中。

## 4. 语文反模式

| 反模式 | 替代做法 |
|--------|----------|
| 默写赤裸裸"请默写XX" | 给使用情境（情境式/理解性默写）|
| 作文题假大空（"以坚持为话题"）| 从小切口进入，贴近学生真实生活场景 |
| 名著选段零碎、不读全书也能答 | 选关键转折点，问题关联前后情节 |
| 文言选文无教材呼应 | 同主题/同人物类型/同文体对照课内 |
| 阅读材料是鸡汤文/说教文 | 选有生活质感的真实文本 |
